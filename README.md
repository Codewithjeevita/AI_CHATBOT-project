# 🤖 AI Chatbot Project

A simple Rule-Based AI Chatbot built using Python.  
This chatbot interacts with users and responds to basic questions using predefined rules.

#🚀 Features
- Greets user based on current time (Morning/Afternoon/Evening/Night)
- Responds to basic questions
- Motivational responses
- Handles simple emotions (happy/sad)
- Exit option using "bye"
- Beginner-friendly project

# 🛠️ Technologies Used
- Python

#▶️ How to Run
1. Open terminal or command prompt
2. Navigate to project folder
3. Run the file using:
   python chatbot.py

#📌 How It Works
- Uses a dictionary to store questions and answers
- Converts user input to lowercase
- Matches input with predefined responses
- Returns response accordingly

# 💡 Example Inputs
- hello
- how are you
- what is python
- motivate me
- bye
